二、提交网址和提交方式
提交网址：https://www.firdc.com/api/member/service_status
请使用 https , 以后不再支持http
提交方式：post



三、参数说明：
当前用户的mid：mid （必须提交，隐藏提交，用户不可见）
当前用户的account：account （必须提交，隐藏提交，用户不可见）
Sign：sign （必须提交，隐藏提交，用户不可见）
Token : token （必须提交，隐藏提交，用户不可见）

四、返回值：

1. 成功时的 返回值示例

{"code":"1","status":"query success","work status":0,"lots":0.01}

work status : 会员交易服务器的工作状态：0.未启动 1.运行中 2.重启中 3.重启失败

lots : 交易的手数
