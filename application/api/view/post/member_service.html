提交方式：post



三、参数说明：
当前用户的mid：mid （必须提交，隐藏提交，用户不可见）
第三方交易平台的帐号account：account （必须提交）
第三方交易平台的密码password：password （必须提交）
第三方交易平台的服务器英文名称trade_server：trade_server （必须提交）
消息messages：messages （必须提交，默认为online，隐藏提交，用户不可见）
会员的交易服务器 英文名称member_server：member_server （必须提交,默认为h02）
交易手数（跟单倍数）：lots （必须提交,默认为0.01）
工作类型：work （必须提交,默认为gj，隐藏提交，用户不可见）
工作状态（0.未启动 1.运行中 2.重启中 3.重启失败） ：work_status （必须提交,默认为0，隐藏提交，用户不可见）
Sign：sign （必须提交，隐藏提交，用户不可见）
Token : token （必须提交，隐藏提交，用户不可见）
