一、会员退出登陆 网页测试表单

mid：

Sign:

二、提交网址和提交方式
提交网址：https://www.firdc.com/api/v1/member/logout
请使用 https , 以后不再支持http
提交方式：post



三、参数说明：
当前会员的mid：mid （必须提交，隐藏提交，用户不可见）
当前会员的sign：sign （必须提交，隐藏提交，用户不可见）
Token : token （必须提交，隐藏提交，用户不可见）

四、返回值：

1. 成功时的 返回值示例

{"code":"2","status":"logout success"} code 为 2 时，当前会员为 退出登陆状态

2. 失败时 返回值示例

{"code":"2","status":"Member already exit!"}
会员已经退出，不要重复提交

{"code":"0","status":"sign error."}
sign 出错

{"code":"0","status":"member's id non-existent"}
会员mid 不存在
