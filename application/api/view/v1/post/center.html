数据说明：

member_profit_rate 会员总盈利率数据
total_all 总盈利百分比
total_year 年盈利百分比
total_month 月盈利百分比
total_week 周盈利百分比
total_today 今天盈利百分比


account_info 帐户信息
account 账户号
deposit 入金
take_profit 出金
profit_amount 盈利金额（盈利）
profit_amount_rate 盈利增长率（盈利率）
total_amount 总金额
server_name 会员交易服务的服务器名称
work_status 状态：0.未启动 1.运行中 2.重启中 3.重启失败 4.取消挂机
