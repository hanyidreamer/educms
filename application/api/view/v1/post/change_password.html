二、提交网址和提交方式
提交网址：https://www.firdc.com/api/v1/member/change_password
请使用 https , 以后不再支持http
提交方式：post



三、参数说明：
当前用户的mid ：mid （必须提交，隐藏提交，用户不可见）
密码：password （必须提交）
Token : token （必须提交，隐藏提交，用户不可见）
Sign : sign （必须提交，隐藏提交，用户不可见）

四、返回值：

1. 成功时的 返回值示例

{"code":"1","status":"change password success","mid":"4","password":"202cb962ac59075b964b07152d234b70"}

2. 失败时的 返回值示例

{"code":"0","status":"member's id non-existent"}
提交的mid 不存在
