二、提交网址和提交方式
提交网址：https://www.firdc.com/api/v1/index/login
请使用 https , 以后不再支持http
提交方式：post



三、参数说明：
用户名：username （必须提交）
密码：password （必须提交）
Token : token （必须提交，隐藏提交，用户不可见）

四、返回值：

1. 成功时的 返回值示例

{"code":"1","status":"login success","mid":"4","username":"cofan","sign":"018c8d3e2aa6e5e3234287092de00264"}
返回值参数说明：
请求返回状态码 code，为1时请求成功，为0时 失败
请求返回状态 status ，login success，登陆成功； login fail 登陆失败
mid 当前用户的mid
username 当前用户的 用户名
sign 当前用户的签名字符串，很重要，请求其他接口都需要带上会员的sign

2. 失败时的 返回值示例

{"code":"0","status":"token error or expired."}
token 错误或过期。

{"code":"0","status":"username is not exist."}
用户名不存在

{"code":"0","status":"login fail,password is error!"}
密码错误
