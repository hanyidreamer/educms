二、提交网址和提交方式

提交方式：post



三、参数说明：


昵称：nickname （必须提交）
手机号：tel （必须提交）
验证码：sms_code （必须提交）
密码：password （必须提交）
Token : token （必须提交，隐藏提交，用户不可见）
